Projects

The Stock Bot

For HackNY, my team decided to explore the Bloomberg API to provide real time stock information in an extremely easy to use way. The Inbox Messanger application provided an excellent platform for us to create a bot that you can message quickly with company names and receive stock data combined with NY Times financial news in response. This project was a great experience learning how to create an application that can effectively communicate with users. For the production version, I transitioned the backend financial source from Bloomberg to Markitondemand. Technologies used included Node.JS, NYTimes API, Bloomberg API, Markitondemand APIs, and Inbox Messanger's API.
You can try out the bot by messaging @thestockbot on Inbox Messanger

Fixey

As a part of a team at CalHacks, we created a web application that uses the Google Directions API to combine bicycle and public transportation data into the most efficient, healthy and eco-friendly way to get around. The fundamental goal behind this project is to encourage people to use alternatives to driving. In addition to providing directions, we informed the users of how much gas money and carbon emissions they were saving by using our suggested route. I built the front end map visualization for Fixey along with assisting with backend processing of the data from Google Maps to GeoJSON. We won the prize for Best Health-Tech Hack. Technologies used included Boostrap, jQuery, Node.JS, Mapbox, D3.JS, and Leaflet

Check it out! Github Repo

New York Minute

I found myself frustrated that there was nothing out there that could simply give me the closest NYC subway stops and tell me when the next trains would arrive. So I decided to make a simple app that does just that! I used MTA's Static Transit Data in conjunction with Node.JS and Parse for handling the data on the backend along with Bootstrap and Javascript/jQuery to provide the output of the results. I am currently in the process of refactoring this project using AngularJS to create a similar application for the University of Michigan Bus System to provide real time bus arrival times based on your location.

Check it out! Github Repo

Live_Code

At MHacks IV, I teamed up with three other great developers to build Live Code, a collaborative coding platform. This application combines video chatting with a shared code editor to allow for users to work together while programming or help a friend out with a code error. We were selected as one of the top 25 teams out of over 250 submissions. I primararily designed on the frontend of the application, but also contributed a few additional features to the backend. Technologies used include Bootstrap, jQuery, EJS, Node.JS, Parse, Socket.IO, WebRTC, SendGrid, and Ace.

Check it out! Github Repo

Recipe Box

As an avid cook, I wanted to make a better way for me to keep track of my favorite recipes online. I built Recipe Box to accomplish this! It's a recipe manager that scrapes a few popular food sites by URL or searches for recipes by ingredient or dish name. Technologies used include Bootstrap, jQuery, EJS, MongoDB, Node.JS, Cheerio, and Yummly's API.
